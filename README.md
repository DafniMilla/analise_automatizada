## ⚠️ Observações Importantes

Os arquivos CSV do SISU podem variar entre os anos (nomes de colunas, estrutura, codificação, etc.).
Por isso, o código pode precisar de adaptações dependendo do arquivo utilizado.
Caso ocorra erro na leitura:

- Verifique o encoding
  
- Confirme o separador (sep)
  
- Ajuste os nomes das colunas
